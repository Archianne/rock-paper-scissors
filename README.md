##  Rock Paper Scissors

Simple game, made with HTML, CSS and Vanilla JavaScript.

### Learn:
- `document.querySelector()`
- `document.querySelectorAll()`
- `document.getElementById()`
- `element.innerHTML`
- `element.addEventListener()`
- `event.target.innerText`
- `Math.random()`
- `if() else if() else`
